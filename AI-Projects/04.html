📘 Practice 4: Advanced Pure HTML

Is practice mein aap kuch naye HTML tags aur attributes seekhenge jo pehle practice mein nahi aaye.

1. Picture Element (Responsive Images)

Yeh element different screen sizes ke liye different images dikhata hai.

Browser ki width kam/karo to image change hoti dikhegi.

2. Inline SVG (Graphics without CSS/JS)

Yeh shapes pure HTML mein draw ki gayi hain (SVG).

3. Definition List (dl, dt, dd)

HTML
HyperText Markup Language – website ka skeleton.
CSS
Cascading Style Sheets – website ki styling.
JS
JavaScript – website ki functionality.

4. BDO (Bi-Directional Override)

Normal text: یہ اردو ہے۔

یہ اردو ہے۔ (RTL override)

یہ اردو ہے۔ (LTR override)

BDO text direction forcefully change karta hai.

5. KBD (Keyboard Input Tag)

Copy karne ke liye dabayein: Ctrl + C

Save karne ke liye: Ctrl + S

6. SAMP (Sample Output)

Computer ka response: Error 404: File not found

7. VAR (Variable Tag)

Equation: x + y = z

8. DFN (Definition Tag)

HTML website ka skeleton hai.

CSS styling ke liye use hota hai.

9. WBR (Word Break Opportunity)

Bohot lamba word jo break ho sakta hai: SuperCaliFragilisticExpialiDocious

Browser ko batata hai ke agar zaroorat ho to yahan break kar sakta hai.

10. Contenteditable Attribute

Yeh paragraph editable hai. Aap isme likh sakte hain!

Is text par click karo aur likhna shuru karo (bina JS ke).

11. Spellcheck Attribute

Yeh paragraph spell check ke saath: Teh quik brown fox jump over teh lazy dog.

Yeh bina spell check ke: Teh quik brown fox jump over teh lazy dog.

Misspelled words underline honge (browser support par depend).

14. Noscript Tag

Agar JS on hai to ye message nahi dikhega.

15. Base Tag (Comment mein hai)

<base> tag head mein likha jata hai. Yeh saare relative links ke liye base URL set karta hai.

16. Meta Tags

Head mein <meta> tags hain: description, keywords, author, charset, viewport.

17. Link Tag (Favicon)

Head mein favicon link hota hai. Yahan example ke liye:

<link rel="icon" href="favicon.ico" type="image/x-icon">
